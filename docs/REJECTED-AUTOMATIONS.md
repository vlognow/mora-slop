## How to Add an Entry

When you evaluate an automation idea and decide not to build it:

1. Add a row to the table above
2. Be specific about **why** — "not worth it" is not specific enough
3. Include whether it's worth revisiting later (and under what conditions)

## Valid Rejection Reasons

- **Too rare** — used less than once a month
- **Too variable** — every instance is different enough that a skill can't capture it
- **Better tool exists** — another tool does this better (name it)
- **Output always needs heavy editing** — the skill doesn't save net time
- **Requires context we can't provide** — needs real-time data, live system access, etc.
- **Premature** — good idea, but we need [X] first
