# Metrics — Department of Vibes

## What We Track

| Metric | How We Measure | Target (30 days) |
|--------|---------------|-------------------|
| Skills contributed | Count in catalog.yaml | 5 |
| Hours saved / month | Self-reported per skill | 5-10 hrs/month total |
| Adoption rate | % of team using ≥1 skill weekly | 50% |
| Quality rate | % of skill outputs sent without editing | 80% |

## How to Report

After using a skill, drop a quick note in `#department-of-vibes`:
- What skill you used
- How long it would have taken manually
- Whether you edited the output (and how much)

We'll compile monthly. No process overhead — just Slack messages.
