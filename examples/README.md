# Examples

Real before/after outputs from skills in this repo. Show don't tell.

## Structure

Each skill gets a subdirectory:

```
examples/
  pm-prd-generator/
    input-1.md      # What the user provided
    output-1.yaml   # Structured output
    output-1.md     # Human-readable output
  meeting-to-jira/
    input-1.md
    output-1.yaml
```

## Why This Matters

New users learn fastest from real examples, not documentation. When you contribute a skill, include at least one real example.
